# Git Workflow Helper

This skill helps you automate common Git operations and follow best practices.

## Features

- **Branch Management**: Create, switch, and manage branches with naming conventions
- **Commit Formatting**: Ensure commits follow conventional commit format
- **PR Assistance**: Generate PR descriptions and templates

## Usage

When working with Git repositories, this skill will automatically:

1. Suggest branch names based on your task
2. Format commit messages appropriately
3. Help create comprehensive PR descriptions

## Example Commands

```bash
# Create a feature branch
git checkout -b feature/add-user-auth

# Create a conventional commit
git commit -m "feat(auth): add JWT token validation"
```
