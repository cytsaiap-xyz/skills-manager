# API Documentation Generator

Generate beautiful, comprehensive API documentation automatically.

## Features

- **OpenAPI/Swagger Support**: Parse and render OpenAPI specs
- **Code Comment Extraction**: Extract JSDoc, docstrings, and godoc comments
- **Interactive Examples**: Generate runnable code examples
- **Multiple Formats**: Output in Markdown, HTML, or JSON

## Usage

1. Point the skill at your API codebase
2. It will scan for API endpoints and documentation
3. Generate formatted documentation

## Output Formats

| Format | Description |
|--------|-------------|
| Markdown | For GitHub wikis and READMEs |
| HTML | For standalone documentation sites |
| JSON | For programmatic access |
